To commit changes the code of that in your computer to your branch on bitbucket: 
0. cd <your downloaded folder> 
1. git status : show the modified files need to be added 
!Note that git status shows which branch are you currently dealing with. If the branch is not your work station, switch by: git checkout <branch_name> 
MAKE SURE YOU ARE ON THE RIGHT BRANCH BEFORE YOU COMMIT YOUR FILES TO BITBUCKET 
2. git add <file> : add the <file> into .git in your computer 
3. git commit -m "message" : commit the changes with the massage to others 
4. git push : upload all changes to the bitbucket
